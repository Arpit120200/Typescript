# Typed To-Do Manager (TypeScript)

## Description
A simple TypeScript-based to-do manager implemented using code-driven inputs.
The project focuses on strong typing, clean function design, and file persistence
without relying on external CLI arguments.

---

## Features
- Add tasks
- List all tasks
- Update task status
- Delete tasks

---
